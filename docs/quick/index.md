## Creating URNs

Create `CtsUrn`s or `Cite2Urn`s directly from strings:

    val ctsUrn = CtsUrn("urn:cts:greekLit:tlg0012.tlg001:1.1")
    val cite2Urn = Cite2Urn("urn:cite2:hmt:msApages.v1:12r")
